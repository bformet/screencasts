constructing DOM elements with D3

* D3 is all about DOM manipulation
* select DOM nodes :
    * `d3.select()`
    * `d3.selectAll()`
* manipulate the selection :
    * `.style()`
    * `.attr()`
    * `.classed()`
    * `.text()`
    * `.html()`
* create / remove nodes :
    * `.append()`
    * `.insert()`
    * `.remove()`
* callback functions :
    * `.each()`
    * `.call()`
